# Downloaded round organisation icons

This pack was rebuilt from downloaded logo/avatar artwork. Source artwork is not redrawn, stretched, recoloured, or stylised. Processing was limited to transparent/empty-margin trimming, aspect-ratio-preserving fit or cover placement, a circular mask, and a subtle circular container stroke.

Contents:

- `png_512/`: 512 × 512 transparent PNG icons.
- `sources/`: downloaded source artwork.
- `manifest.json`: source URL, source page, processing mode, and notes per organisation.
- `preview_contact_sheet.png`: visual overview.

The logos remain subject to their respective owners’ trademark/brand rules.
